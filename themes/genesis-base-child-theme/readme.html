Theme Information

This theme is based on the Genesis Framework using a custom starter theme built by Whiteley Designs. For details on the structure of this theme please visit the wiki (https://github.com/mwhiteley16/Whiteley-Designs-Blueprint/wiki/Base-Starter-Theme-Wiki)

Code Compiling

This theme uses SASS for all CSS output and CodeKit to compile. For access to the CodeKit config file please email me: matt@whiteleydesigns.com
